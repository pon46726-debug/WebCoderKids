📄 HTML

Основы HTML

HTML (HyperText Markup Language) — это скелет веб-страницы. Он создаёт структуру: заголовки, параграфы, списки и многое другое.

📄 Основы HTML

HTML — это язык разметки, который создаёт структуру веб-страницы.

<!DOCTYPE html> <html> <head> <title> Моя первая страница </title> </head> <body> <h1> Привет, мир! </h1> <p> Это мой первый веб-сайт. </p> </body> </html>

Что это значит?

1
<!DOCTYPE html>

Говорит браузеру, что это HTML5 документ

2
<html>

Корневой элемент всей страницы

3
<head>

Содержит мета-информацию (заголовок, стили)

4
<body>

Всё видимое содержимое страницы

5
<h1>

Заголовок первого уровня (самый большой)

6
<p>

Абзац текста

Т Теги текста
<!-- Заголовки разных уровней --> <h1>Заголовок 1</h1> <h2>Заголовок 2</h2> <h3>Заголовок 3</h3> <!-- Форматирование текста --> <p>Это <strong>жирный</strong> текст</p> <p>Это <em>курсив</em></p>
☰ Списки
<!-- Маркированный список --> <ul> <li>Первый пункт</li> <li>Второй пункт</li> </ul> <!-- Нумерованный список --> <ol> <li>Шаг первый</li> <li>Шаг второй</li> </ol>
</> Ссылки и картинки
<!-- Ссылка --> <a href="https://example.com"> Нажми сюда! </a> <!-- Картинка --> <img src="photo.jpg" alt="Описание">
🎨 CSS

Стили CSS

CSS (Cascading Style Sheets) отвечает за внешний вид. С его помощью можно менять цвета, шрифты, расположение элементов и создавать анимации!

🎨 Цвета и фон

CSS делает страницу красивой! Добавляй цвета и фоны.

HTML:

<div class="box"> Привет, мир! </div>

CSS:

.box { background-color: #8b5cf6; color: white; padding: 20px; border-radius: 10px; text-align: center; font-size: 24px; }

✎ Стили текста
h1 { font-family: 'Arial', sans-serif; font-size: 48px; color: #333; text-shadow: 2px 2px 4px rgba(0,0,0,0.3); text-transform: uppercase; letter-spacing: 2px; }
🗐 Блочная модель
.box { width: 300px; height: 200px; padding: 20px; border: 5px solid #8b5cf6; margin: 20px auto; box-shadow: 0 10px 30px rgba(0,0,0,0.2); }
☰ Flexbox
.container { display: flex; justify-content: center; align-items: center; gap: 20px; flex-wrap: wrap; }
⚡ JavaScript

Интерактив с JavaScript

JavaScript добавляет магию на сайт! Он делает страницы интерактивными: реагирует на клики, анимирует элементы и может менять всё в реальном времени.

📦 Переменные

Переменные хранят данные: числа, текст и многое другое.

// Создаём переменные let name = "Алиса"; let age = 12; let isStudent = true; // Выводим в консоль console.log("Привет, " + name); console.log("Тебе " + age + " лет"); // Меняем значение age = 13; console.log("Теперь тебе " + age);
